ğŸ“‹ QUICK REFERENCE

To start development:
  1. Edit .env.development with your database details
  2. Run: uv sync
  3. Run: uv run uvicorn app.main:app --reload
  4. Visit: http://localhost:8000/docs

To add a new field to users:
  1. Edit app/models/users.py (add to User class)
  2. Add to UserBase, UserCreate, UserUpdate, UserPublic schemas
  3. Create migration or restart (tables auto-created on startup)

To protect an endpoint:
  1. Import require_role or require_permission from app.core.security
  2. Add to endpoint dependencies
  3. See app/api/auth.py for examples
